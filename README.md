#  Zomato Clone

A responsive Zomato-inspired food delivery website built using HTML, CSS, and JavaScript.

##  Features

- Responsive Navigation Bar
- Hero Section with Search Bar
- Popular Cuisines Section
- Restaurant Cards
- More Restaurants Section
- Why Choose Us Section
- Back To Top Button
- Scroll Animations
- Modern UI Design

##  Technologies Used

- HTML5
- CSS3
- JavaScript

## 💻 Installation

1. Clone the repository

```bash

```

2. Open project folder

```bash
cd zomato-clone
```

3. Open index.html in browser
